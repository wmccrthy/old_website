Projects

Personal projects I've done outside of class. Languages used:

Pathfinding Visualizer

My pathfinding visualizer supports breadth-first search, depth-first search, Dijkstra’s algorithm, and A* search; users can run these algorithms on variations of the initial grids, which can be modified by adding walls and weighting nodes.

Checkers AI

A checkers game that supports 2 player or 1 player use. The CPU opponent uses the minimax or expectiminimax algorithms. I spent extensive time optimizing the minimax algorithm using various techniques such as alpha-beta pruning and transposition tables.

Sorting Visualizer

Very simple program to display a visualization of how various sorting algorithms work. Mastered the logic of these algorithms such that the program is able to not only demonstrate an algorithm’s final product but how it produces such a result on an iterative and/or recursive level.

Nescac Stats

Java program that reads an excel file with data from every varsity soccer player in the New England Small College Athletic Conference. The program computes stats for each individual and team in the conference using common metrics and provides an interface to access said data. This was one of my earliest projects when I was first learning Java.
